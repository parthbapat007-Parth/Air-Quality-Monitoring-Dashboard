# 📖 Overview

The **Air Quality Monitoring Dashboard** is a modern web application that provides users with **real-time Air Quality Index (AQI)**, weather information, pollutant analysis, health recommendations, and weather forecasts using live APIs.

The dashboard is designed with a responsive user interface and interactive charts to help users understand the environmental conditions around their location or any searched city.

---

# ✨ Features

- 🌍 Real-Time Air Quality Index (AQI)
- 📍 Automatic Location Detection
- 🔍 Search Air Quality by City
- 🌡 Live Temperature
- 💧 Humidity Monitoring
- 🌬 Wind Speed
- ☀ UV Index
- 🌦 Weather Forecast
- 📊 Interactive AQI Charts
- 🧪 Pollutant Analysis
- ❤️ Health Recommendations
- 📱 Fully Responsive UI

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/parthbapat007-Parth/Air-Quality-Monitoring-Dashboard.git
```

## Navigate into Project

```bash
cd Air-Quality-Monitoring-Dashboard
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```
